## Demo

Access to https://keywoo.herokuapp.com/

## Requirements

- docker >= 19.03
- docker-compose >= 1.24

## Installation

```
git clone git@github.com:ymmmtym/keywoo.git
docker-compose up -d --build
```

Access to http://localhost:5000
